# Changelog

All notable changes to this project will be documented here.

## 0.1.0 – MVP
- Journal: create/edit daily entries (mood, pain, fatigue, nausea), notes, tags
- History: filter, edit, delete (with confirm)
- Trends: 7/30/All range, smoothing Off/7/30-day
- Export: JSON/CSV; Import JSON with conflict resolution
- Backups: periodic reminder + one‑click JSON backup
- PWA: manifest + service worker, icons (SVG; optional PNGs)
- Accessibility and UX polish (fieldset/legend, aria labels, toasts)
